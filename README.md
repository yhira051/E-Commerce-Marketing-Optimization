# E-Commerce Marketing Optimization using Uplift Modeling (T-Learner)

## 1. プロジェクトの背景と目的
従来の機械学習を用いたマーケティング施策（例：購買予測モデル）では、「放っておいても買う顧客（鉄板層）」と「アプローチすれば買う顧客（説得可能層）」を区別できず、無駄なコスト（カニバリゼーション）を発生させてしまう課題がありました。
本プロジェクトでは、因果推論のアプローチである**アップリフトモデリング（Uplift Modeling）**を導入し、顧客ごとの「純粋な施策効果（因果効果）」を予測しました。これにより、「メールを送ることで初めて購買行動を起こす顧客」のみをターゲティングし、マーケティングROI（投資対効果）を最大化することを目的としています。

## 2. 使用データ
実際のEコマースサイトにおけるメールマーケティングキャンペーンのA/Bテスト（ランダム化比較試験：RCT）データを使用しました。

- **サンプルサイズ**: 64,000件
- **介入（Treatment）**:
  - `Treatment = 0`: コントロール群（メール配信なし）
  - `Treatment = 1`: 男性向け商品メール配信群
  - `Treatment = 2`: 女性向け商品メール配信群
- **目的変数（Outcomes）**:
  - `visit`: サイト訪問有無（二値分類）
  - `conversion`: 商品購入有無（二値分類）
  - `spend`: 購入金額（回帰）
- **特徴量（Features）**: `recency`（最終購入からの月数）, `history`（過去の購入金額）, `newbie`（新規顧客フラグ）など。

## 3. 分析アプローチ（T-Learnerの実装）
アップリフトスコア（個人の因果効果）を算出するため、メタ学習器の一種である**T-Learner (Two-Learner)**を採用しました。ベースとなる機械学習アルゴリズムにはRandom Forestを使用しています。

### モデリングの手順
1. **学習フェーズ**: 介入群（メールあり）とコントロール群（メールなし）のデータを分割し、それぞれ独立したRandom Forestモデルを学習させます。
2. **予測フェーズ**: 全顧客データに対して、両方のモデルから予測値を出力させます。
3. **スコア算出**: 以下の計算式により、アップリフトスコアを算出しました。
   `Uplift Score = (介入群モデルの予測値) - (コントロール群モデルの予測値)`
   ※ スコアが正の大きい値であるほど、「メールによる押し上げ効果が高い（説得可能層）」と判定されます。

## 4. モデルの評価（Uplift Curve）
算出したスコアが本当に因果効果を捉えられているかを検証するため、**アップリフトカーブ（Uplift Curve）**を描画して評価を行いました。

- **評価ロジック**: スコアの降順に顧客を並べ、上位から累積の「介入群の平均成果 - コントロール群の平均成果」を計算。
- **結果**: ランダムターゲティング（ランダムベースライン）と比較して、序盤でカーブが急激に立ち上がり、高いAUUC（Area Under the Uplift Curve）を確保しました。これにより、モデルが効果の高い顧客を正しく上位にランク付けできていることが証明されました。

## 5. ビジネスインパクト（利益シミュレーション）
実務導入時の効果を測定するため、メール1通あたりの送信コスト（または機会損失）を $0.1 と仮定し、従来の「一斉送信」と「AIターゲティング配信（Score > 0 のみに送信）」でROIの比較シミュレーションを実施しました。

| キャンペーン | 配信シナリオ | 配信数 | 純増売上 | コスト | 利益 | 改善額 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **男性向け** | 全員配信 | 42,613 | $32,804.64 | $4,261.30 | $28,543.34 | - |
| | **AI配信 (Score>0)** | 39,628 (93%) | $52,618.59 | $3,962.80 | $48,655.79 | **+$20,112.45** |
| **女性向け** | 全員配信 | 42,693 | $18,119.43 | $4,269.30 | $13,850.13 | - |
| | **AI配信 (Score>0)** | 32,729 (76%) | $39,422.14 | $3,272.90 | $36,149.24 | **+$22,299.11** |

**結論**: 無駄な配信（鉄板層・無関心層）や逆効果になる配信（天邪鬼層）を回避することで、配信コストを削減しつつ純増売上を大きく引き上げ、**合計で約$42,000の利益改善**を実現しました。

## 6. 要因分析による顧客インサイト
モデルをブラックボックスにせず、AIが「説得可能」と判断したターゲット層（Score > 0）と非ターゲット層（Score <= 0）の属性を比較分析し、マーケティング戦略に直結するペルソナを抽出しました。

- **男性向けキャンペーンのペルソナ**:
  「直近で購買行動があり、すでに自社で高い金額を使っている（Historyが高い）優良顧客および新規顧客」。鉄は熱いうちに打つアプローチが有効。
- **女性向けキャンペーンのペルソナ**:
  「過去の累積購入金額がまだ低く、新規顧客（Newbie）の割合が高い層」。既存のヘビーユーザーはメールがなくても買うため除外し、新規開拓にリソースを割くのが有効。
